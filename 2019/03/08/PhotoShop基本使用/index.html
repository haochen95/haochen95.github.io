PhotoShop基本使用

发表于 2019-03-08 | 分类于 前端开发 | 评论数： | 阅读次数： | 阅读次数：

[TOC]

1. 图片格式

格式 说明 用途
psd photoshop的专用格式，这种格式是不压缩的，保留了突出，透明和半透明信息 保存图片原始数据，方便图片修改
jpg 有损压缩，压缩效率高，容量小，网络传输快,不可保存为透明背景 不需要透明背景的时候使用
gif 无损压缩格式，最多有256种颜色，颜色丰富的图片转化为这种格式会颜色失真，它的背景也可以是透明，有锯齿 也可以制作动态
png 代替gif，无损压缩，背景可以是透明和半透明，没有锯齿 用于firework处理图片的
webp 压缩方面比jpg更加优秀，在质量相同的情况下，webp格式的体积比jpg小40% 只在Chrome上面支持
位图和矢量图 由一个个的方形的像素点排列在一起拼接，位图放大时，图像会失真，上面的5种都是位图
矢量图是由函数曲线组成的，无图像块
svg 矢量二维图格式，即使放大也不会有方块出现或者失真 但是颜色不丰富
flash 矢量动画文本格式 现在少用了

总结

  1. 如果使用不透明背景的图片，用jpg
  2. 如果使用透明或者半透明的图片,用png
  3. 制作网页图标时候，如果图标有多种颜色，用gif/png，如果图片是单色的用svg，如果图标是动画的用gif

2. PS的基本操作

2.1. 工具栏介绍

选择图层

  1. 点击移动工具
  2. 按住Ctrl键选择某个图层

图像放缩平移

  1. 缩放工具： 按住alt点击缩小，快捷键ctrl + +/-来控制放大缩小，双击缩放工具图标恢复到原来的大小
  2. 平移工具： 对图像进行平移，如果在使用其他工具的前提下，直接按住空格即可实现平移

移动选择和图像

  1. 按住Ctrl，在图像上面点击可以选中图层
  2. 选择移动工具，勾选工具属性栏上的自动选择图层，可以在图像上点击选中的图层
    1. 如果操作失误，可以通过历史记录回到之前的状态
    2. 一般不建议自动选择，取消后，按住Ctrl键就可以选择某个选中的图层
  3. 移动元素同时按住alt键可以复制移动一个图层
  4. 图层面板的操作，包括图层的显示隐藏，图层顺序，新建图层，图层删除等

针对图像中选择图层的操作

  1. 移动
  2. 自由变换(Ctrl + T)(Shift等比例放缩)，执行菜单命令， 编辑
  3. 拖拽到另外一张图片上面完成图层拷贝

选框工具

  1. 有三种样式： 矩形，椭圆形，单行，单列
  2. 有三种选区模式：新选区，添加选区，从选区减去，选区叠加
  3. 可通过Ctrl+D取消选区

套索工具

  1. 套索工具
    • 自己画出来的选区，其他操作跟选框工具一样
  2. 多边形套索工具
    • 通过多边形线段连接来构建选区(delete键删错错误操作)
  3. 磁性套索工具
    • 顺着轮廓移动，自动产生

魔棒和快速选择工具

  1. 在需要选择的区域内部点击
  2. 配合添加选区使用
  3. 魔棒工具跟上面的功能类似，但是需要调整容差和连续性
  4. 魔棒+反选：最强大

* 针对选区的操作

  1. 复制
  2. 粘贴
  3. 填充(编辑->填充)
  4. 描边（编辑-描边）：对边框设置
  5. 删除-delete键
  6. 自由变化Ctrl+T(可以改变选区的大小)

擦除和污点

修改错误

文本输入

  1. 选择T工具
  2. 再选择工具-》放在需要修改的文本上-》修改文本

参考线

  1. 先打开：视图-》标尺
  2. 然后打开：视图-》新建参考线-》输入位置建立水平/垂直的参考线

图像大小

  1. 点击：图像-》图像大小：查看图像的尺寸大小

尺寸测量

  1. 切片工具： 双击切片弹出切片对话框
  2. 切片选择工具
  3. 矩形框工具

裁剪图像

  1. 对选区执行图像裁切

2.2. 批量切图

在一张效果图中，同时切多个图

使用切片工具

  1. 先切一个，切完一个，双击后命名保存
  2. 一定要把背景藏起来后(图层)，然后再保存
  3. 文件->存储为web存储格式->选择PNG-24

2.3. 雪碧图

将一些小图标合并成一个图，发送HTTP请求，用于优化网页加载 ，适用于装饰性图片，比如说图标，箭头等

  1. 新建一个大致高宽的图，背景为透明
  2. 根据使用的顺序，从文件夹拖入到创建的图中(上下顺序)
  3. 创建50px等高的参考线
  4. 将每个图片放在左上角
  5. 然后只选择图片区域，图像->裁剪->保存为PNG-24格式

3.前端开发流程

  1. 创建项目目录
    • 创建image, css, js文件夹
  2. 使用PhotoShop对效果图进行切图，切出网页制作中需要的小图片
  3. 将装饰类图像合并，制作成雪碧图
  4. 结合PhotoShop和代码编辑器，参照效果图，进行HTML和CSS代码书写，制作页面
